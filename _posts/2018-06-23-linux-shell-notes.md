## wget
```bash
wget www.xxx.xxx/file.txt
wget www.xxx.xxx/file{000..100}.txt
wget -i urllist.txt
```

## curl
```bash
curl -O www.xxx.xxx/file.txt
curl -O www.xxx.xxx/file[000-100].txt
```
## qpdf
```bash
qpdf --decrypt infile.pdf outfile.pdf
```
